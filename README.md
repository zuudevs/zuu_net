Tema : Infrastruktur TI Korporat \& Keamanan Siber
Judul : Grid-Watcher
Deskripsi : Monitoring \& Mitigasi Serangan Siber Real-time pada Jaringan SCADA Menggunakan Algoritma Native C++ Low-Latency


Tujuan :

* Efisiensi Sumber Daya pada Infrastruktur Kritis (The "Low-Latency" Point)
* Otomatisasi Mitigasi Serangan (The "Self-Healing" Point)
* Akurasi Deteksi Berbasis Perilaku (The "Behavioral Analysis" Point)
* Kemandirian Teknologi \& Keamanan Native (The "Native C++" Point)



Detail Tujuan :

* Mengembangkan sistem keamanan jaringan yang sangat ringan (ultra-lightweight) dan efisien, sehingga dapat diimplementasikan pada perangkat keras terbatas (low-spec hardware) atau komputer lawas di lingkungan operasional gardu induk tanpa membebani kinerja CPU dan memori utama.
* Mewujudkan mekanisme pertahanan aktif (Active Defense) yang mampu mendeteksi anomali trafik dan memblokir alamat IP penyerang secara otomatis (Self-Healing) dalam hitungan milidetik, meminimalisir keterlambatan respons yang sering terjadi pada pemantauan manual.
* Mengimplementasikan algoritma analisis perilaku (Behavioral Analysis) untuk membedakan trafik operasional yang sah (seperti komunikasi VoIP/Microsoft Teams dan sinkronisasi data SCADA) dengan serangan DDoS/Flooding, guna menekan tingkat kesalahan deteksi (False Positive).
* Mendemonstrasikan rancang bangun keamanan siber yang mandiri menggunakan Native C++ Raw Sockets, meniadakan ketergantungan pada kerangka kerja (framework) pihak ketiga yang berisiko memiliki celah keamanan bawaan atau overhead performa yang tinggi.
